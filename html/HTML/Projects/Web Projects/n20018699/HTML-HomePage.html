Density
Objects with the same weight have the same mass.
A single feather does not have the same mass as a gold brick because it does not have the same density. To find the density you must divide the mass by the volume.

Vocabulary

  • Mass- The amount of matter in an object.
  • Volume- How much space an object takes up.
  • Density- How compact an object's matter is.
