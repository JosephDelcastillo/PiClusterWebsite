Shortcuts might seem hard to remember at first and like you might never actually use them but once you get into the habit of using them you'll naturally start remembering and using them, sometimes without even realizing it, who knows, maybe one day you might get so used to a shortcut it might feel weird to not use it.

Shortcut Action
Alt + Tab Quick Switch to Your Most Recent Window
WindowsKey + Tab Shows you all Open Windows at Once for Easy Switching
WindowsKey + Arrow Key Allows You to Snap the Window Into a Corner or Side of the Screen
WindowsKey + E Opens the File Explorer
WindowsKey + I Opens the Settings Window
WindowsKey + L Locks Your Computer
WindowsKey + R Opens the Run Box
WindowsKey + S Opens System Search Bar
WindowsKey + TopRowNumber Allows You to Switch to the Corrisponding Program in the Taskbar
WindowsKey + Comma Allows You to Peek at the Desktop
WindowsKey + "+" or "-" Allows You to Zoom-In or Zoom-Out Using the Windows Magnifier
Ctrl + E (In Chrome) Allows You to Use Your Default Search Engine From Anywhere in the Browser
